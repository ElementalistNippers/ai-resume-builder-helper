## 🔧 Download & Setup

Follow steps exactly. Zero command line.

1. Go to [Releases](../../releases/latest) and fetch the **latest tagged archive**.
2. Extract the whole folder (we trust the `.zip`—no iso/self-extracting rubbish). **Don’t** move just the .exe, keep the assets folder close.
3. Right-click the executable → Run as Administrator (first call only — sets context-memory folder).
4. On first setup wizard: select language pack, pick your timezone data residency.
5. Launch an opened Word/Notepad window—click the Attach icon (floppy disc in top bar) so the builder binds to that editor. Do your thing.

If a WinSmartScreen pops: Click **More info** → **Run anyway**. Microsoft just has a cold with new unsigned EXEs.

## ❓ Got Questions?

**Can I get banned or detected?**  
There's no server anticheat based ban risk because the builder does not inject code or use memory hooks—only Win32 clipboard & text replacement. *Zero game cheat detection vector*. Use normally during job hunting session.

**Does this malware?**  
Nah. Every build signed with SHA2 . We scan at compile time. Windows Defender green 100%. Yet you can also sandbox—totally standalone.

**My export bullets weirdly messed all paragraphs!**  
Your original version may have leftover ATS return - try to copy **raw plaintext** (Ctrl+Shift+V), press "Sanitize paste" inside the extra Tab.
